Architecture Details

Scheduler
GitHub Actions Cron
Runs Mon / Wed / Fri at 9am CST
Generates 1 post per run
~3 posts/week cadence
Data Sources
Reddit + X
Reddit: r/sales, r/salesforce, r/coldcalling, r/B2Bsales
X: keywords around meeting follow-ups, sales tools, CRM pain
AI Generation
Claude API + Your Voice
Pulls voice prefs from user_preferences table
Alternates problem-solution & comparison formats
Generates title, slug, excerpt, content, tags, reading time
Delivery
GitHub PR + Slack
Auto-creates branch & PR with new post in blog-data.ts
Posts to #blog-drafts Slack channel with title, excerpt, PR link
Merge PR = publish to site

What Gets Built

new .github/workflows/blog-agent.yml — cron schedule + orchestration
new scripts/blog-agent/scrape.ts — Reddit + X scraping
new scripts/blog-agent/analyze.ts — pain point extraction + keyword analysis
new scripts/blog-agent/generate.ts — Claude draft generation with voice prefs
new scripts/blog-agent/publish.ts — create branch, update blog-data.ts, open PR
new scripts/blog-agent/notify.ts — Slack webhook notification
mod lib/blog-data.ts — new posts appended by agent PRs
